## Overview

The Prisma client lets you send raw database queries using the `$raw` property:

```ts
prisma.$raw.__DB_NAME__(query: String): Promise<Array<any>>
```

`__DB_NAME__` is a placeholder for the name of a database that is connected to your Prisma server. For each Prisma server, there is one `default` database - for convenience, you can omit the dot notation for the `default` database: 

```ts
// the following two lines are equivalent and both access the `default` database
prisma.$raw.default(query: String): Promise<Array<any>>
prisma.$raw(query: String): Promise<Array<any>>
```

You can find more info about this topic [here](https://github.com/prisma/prisma/issues/2052).

### Examples

```js
prisma.$raw("SELECT * FROM user LIMIT 1")
```
