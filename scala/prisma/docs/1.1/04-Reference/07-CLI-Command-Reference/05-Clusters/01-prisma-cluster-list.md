# `prisma cluster list`

List all [clusters](!alias-eu2ood0she). This includes the cluster from your [cluster registry](!alias-eu2ood0she#cluster-registry) as well as the ones you have configured through Prisma Cloud.

#### Usage

```sh
prisma cluster list
```

#### Examples

##### List all clusters.

```sh
prisma cluster list
```
