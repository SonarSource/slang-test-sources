# `prisma cluster add`

Add a new cluster to the [cluster registry](!alias-eu2ood0she#cluster-regsitry). This command triggers an interactive prompt where you need to provide the _name_, `host` and `clusterSecret` for the new cluster. 

#### Usage

```sh
prisma cluster add
```
